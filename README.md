## Code-Friend: 
本项目采用 VuePress , 一个基于 Vue SSR 的静态站生成器。搭建一个可以自动解析Markdown文档的文档库。

[在线文档地址](http://doc.friendlycode.cn)

## 勘误及提问
如果有疑问或者发现错误，可以在相应的 issues 进行提问或勘误。

如果喜欢本项目或者对您有所启发，欢迎 star, 对作者也是一种鼓励!
